# CUT/CTRL

Contemporary editorial barbershop experience built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, Lucide and Next Image.

## Design direction

- Warm off-white field, near-black structural grid, electric cobalt actions.
- Condensed display typography with terse grotesk utility copy.
- Square geometry, shared border system, asymmetric image crops, minimal shadow.
- Seven signature interaction families: clipped reveal, marquee, shared service preview, recommendation transitions, compare slider, expanding crew rail, accessible lightbox/favorites; the booking stepper is reserved for transaction clarity.

## Routes

`/`, `/cuts`, `/crew`, `/lookbook`, `/book`, `/studio`, plus loading, 404, sitemap and robots.

## Run

```powershell
npm.cmd install
npm.cmd run dev
```

The included runner uses Next.js WebAssembly bindings plus Webpack on this Windows environment because its downloaded native SWC binary is unavailable.
